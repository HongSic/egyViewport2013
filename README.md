## Setting

You can change settings via `config.json` or (and) `background.png`.

Example of `config.json`:
```json
{
  "imageFileName": "background.png",
  "imageOpacity": 0.2
}
 
```

- `imageFileName`: **string** file name
- `imageOpacity`: **double** image opacity (must be `0` to `1`)
